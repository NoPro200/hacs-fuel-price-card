# Changelog

## [1.0.0] - 2026-03-11

First public release.

### Features
- Multiple gas stations per card
- Multiple fuel sensors per station with custom labels
- Trend arrows via Home Assistant History API (30-day lookback)
- Sort stations by lowest price with cheapest badge
- Click fuel chip to open HA more-info / history dialog
- Full visual UI editor with collapsible station blocks
- Smart sensor filter: fuel keywords + price range 0-10 EUR
- Station logo support via URL
- Drag-and-drop reordering of stations and sensors
- Backward compatible with old single-station YAML format
- Live card preview in the card picker
